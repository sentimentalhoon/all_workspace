# PSMO Community Frontend

Vue.js 3 기반 커뮤니티 플랫폼 프론트엔드

## 기술 스택

- Vue.js 3.5+
- Vue Router 4
- Pinia (상태 관리)
- Vite 6
- Axios

## 설치 및 실행

### 의존성 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npm run dev
```

### 프로덕션 빌드

```bash
npm run build
```

### 빌드 미리보기

```bash
npm run preview
```

## 포트

- 3000 (개발 서버)

## API 프록시

개발 환경에서 `/api` 경로는 `http://localhost:8080`로 프록시됩니다.

## 주요 기능

- 사용자 인증
- 커뮤니티 게시판
- 프로필 관리
- 실시간 알림
